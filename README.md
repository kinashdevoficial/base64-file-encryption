# base64-file-encryption
Base64 file encryption with Python 2.7

## How to use
**To Encrypt**

Just tip in your terminal `python encrypt.py file_to_encrypt.py`

**To Decrypt**

Just tip in your terminal `python decrypt.py file_to_decrypt.py`
